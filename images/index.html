A Bluetooth Smartwatch

Conception and building of a bluetooth smartwatch which was of reasonable size. Built using an AtMega processor, a bluetooth chip, a 3,7V lithium battery, a small vibrator, a charging module as well as an OLED screen. The case was 3D printed using SketchUp to design it. A windows app was coded using C# to send notifications via bluetooth to the watch and sync time. The watch would then vibrate and show the notification content on the screen. The watch can charge using USB, and has a battery life of around one day.
